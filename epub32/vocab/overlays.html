メディア オーバーレイのメタデータ語彙

概要

この語彙について

この語彙のプロパティは、meta 要素の property 属性 [[Packages32]] で使用できる。

参照

この語彙を参照するためのベース IRI は、http://www.idpf.org/epub/vocab/overlays/# である。

プリフィックス「media:」は、この語彙にあるプロパティを持つ [[!Packages]] で使用するために予約済みであり、パッケージ ドキュメントで宣言する必要はない。

メディア オーバーレイ プロパティ

active-class

名前： active-class
概要： 現在再生中の EPUB コンテンツ ドキュメントの要素に適用する製作者定義の CSS クラス名。
許可された値： xsd:string
個数： 0 又は 1
例： <meta property="media:active-class">-epub-media-overlay-active</meta>

duration

名前： duration
概要： プレゼンテーション全体又は特定のメディア オーバーレイの持続時間。指定の持続時間は、オーサリング時に知られている音声クリップを考慮し、外部リソース又は合成音声のライブ ストリーミングは除く。
許可された値： クロック値。

[[!SMIL3]] clock value でなければならない（MUST）。
個数： 既知のレンディション及び各メディア オーバーレイに対して厳密にひとつ。
例： <meta property="media:duration">1:36:20</meta>

narrator

名前： narrator
概要： ナレーターの名前。
許可された値： xsd:string
個数： 0個以上
例： <meta property="media:narrator">Joe Speaker</meta>

playback-active-class

名前： playback-active-class
概要： 再生中に、EPUB コンテンツ ドキュメントのドキュメント要素に適用する製作者定義の CSS クラス名。
許可された値： xsd:string
個数： 0 又は 1
例： <meta property="media:playback-active-class">-epub-media-overlay-playing</meta>
